# MediFit

MediFit is a comprehensive digital platform designed to streamline workout tracking and fitness management. Built with a focus on user experience and performance, it allows users to log exercises, monitor progress, and manage their health journey in one place.

## 🚀 Features

- **Workout Logging:** Easily track your daily exercises, sets, and repetitions.
- **Progress Tracking:** Visualize your fitness journey through intuitive data displays.
- **Customizable Routines:** Create and save workout plans tailored to your specific fitness goals.
- **Responsive Design:** Fully optimized for a seamless experience across desktop and mobile devices.

## 🛠️ Tech Stack

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend/Database:** Supabase
- **Deployment:** Vercel
- **State Management:** TanStack Query (React Query)
- **UI Components:** Lucide React for iconography

## 🏁 Getting Started

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
